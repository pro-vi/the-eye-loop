**Source of truth for new patterns:** `specs/4-akinator.md`

---

## Section 1: Anima YAML Serialization → REPLACE

The entire axis-based YAML format (resolved/exploring/unprobed/anti_patterns) is dropped.

**Replace with evidence serialization:**

```
# Evidence | {swipeCount} swipes
intent: "{user's original input}"

1. [ACCEPT] "{content}"
   Hypothesis: {hypothesis}

2. [REJECT (hesitant)] "{content}"
   Hypothesis: {hypothesis}

anti_patterns:
  - {extracted from rejects by builder}
```

- No axes, no confidence, no distributions
- `(hesitant)` tag = slow latency = near decision boundary = most informative
- Anti-patterns still exist — builder extracts them from rejections
- Between-compaction code updates: just push to evidence array (no distribution math)

## Section 2: Scout System Prompt → REPLACE

Drop the axis-targeting prompt. Replace with validated Akinator prompt from `specs/4-akinator.md`:

**Key changes:**
- `ANIMA STATE: {anima_yaml}` → `EVIDENCE HISTORY: {evidence_list}`
- Remove: "Target the EXPLORING dimension with the flattest distribution"
- Remove: "Every RESOLVED dimension is LOCKED"
- Remove: `STAGE:` field and stage rules — scout chooses format based on evidence depth
- Remove: `dimension:` from output metadata
- Remove: `held_constant:` from output metadata
- Add: "Think like Akinator — each question should maximally partition the remaining possibility space"
- Add: "Do NOT re-confirm things we already know"
- Add: "Target the GAPS"

**LOCAL HISTORY stays** but drop `dimension:` field. Keep: facade_id, hypothesis, decision, latency_signal, lesson.

**IMAGE SCHEMA stays unchanged** — still the right format for image-stage facades. But scout chooses WHEN to use it based on evidence depth, not a stage flag.

## Section 5: Oracle Session-Seed → DROP ENTIRELY

The entire axis-seeding prompt is eliminated. No LLM call on session init. The scout generates its first probe from just the intent string. Validated in bench-akinator H1 Round 1 (6/6 quality from intent alone).

Session init becomes pure code: `context.reset(); context.intent = intent; context.sessionId = randomUUID();`
