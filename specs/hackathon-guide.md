# Zero to Agent: Vercel x Deepmind Hackathon SF Participant Guide

## 1. Problem Statements

Must build in at least one:

- **Statement One: Chat-Based Agents** — Using Vercel's ChatSDK, create agents that plug into platforms like Slack, Discord, Teams, WhatsApp, GitHub, or Linear. Write bot logic once and deploy across multiple platforms. Bonus for Gemini multimodal capabilities.

- **Statement Two: Multi-Modal Agents** — Leverage Gemini's multimodal capabilities to create agents that see, hear, or understand the world through images, video, audio, documents, or any combination.

- **Statement Three: AI Applications** — Build an AI-powered application that solves a real problem. Doesn't have to be fully autonomous — just useful, creative, and well-built. Focus on craft, user experience, and shipping something polished.

**We are building Statement Three.**

## 3. Rules

- **Open Source:** Repositories must be public.
- **Team Size:** Maximum of four. Solo allowed.
- **Demo Requirements:** Must only highlight features, code, and functionality built during the hackathon. Judges must clearly identify what was created during the event. Failure = disqualification.
- **New Work Only:** May not present existing project as own work. Failure = disqualification.
- **Banned:** Projects that violate legal, ethical, or platform policies, use code/data/assets without rights.

### Anti-Projects (STRICTLY NO):
- AI Mental Health Advisor
- Basic RAG Applications
- Streamlit Applications
- Image Analyzers
- "AI for Education" Chatbot
- AI Job Application Screener
- AI Nutrition Coach
- Personality Analyzers
- Any project using AI to generate/give medical advice

### Round One (assigned groups, ~3 min demo + 1-2 min Q&A):
- Impact Potential (20%) — Long-term potential, lasting impact, usefulness beyond hackathon
- Live Demo (45%) — Core idea implementation, does it work live, presentation quality
- Creativity and Originality (35%) — Novel concept, differentiation, unique approach to problem statements

- Equal weighting for each category
